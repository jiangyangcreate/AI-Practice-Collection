# 示例数据目录

这个目录用于存放测试用的 PDF 文档。

## 使用方法

1. 将您的 PDF 文件放入此目录
2. 在应用中上传这些文件进行测试

## 获取示例 PDF

您可以使用任何 PDF 文档进行测试，例如：

- 技术文档
- 研究论文
- 用户手册
- 电子书

## 示例文档推荐

如果您需要测试文档，可以从以下来源获取：

1. **arXiv**: https://arxiv.org/ - 学术论文
2. **Google Scholar**: https://scholar.google.com/ - 研究文献
3. **自己的文档**: 任何 PDF 格式的文档都可以

## 注意事项

- 确保 PDF 文件包含可提取的文本（非纯扫描件）
- 建议文件大小不超过 50MB
- 文件名尽量使用英文，避免特殊字符
